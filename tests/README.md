# Tests Directory

## HEADER
- **Purpose**: Organize the automated test suite for this project
- **Status**: Active
- **Date**: 2025-11-28
- **Dependencies**: Project source code and test frameworks
- **Target**: Developers and CI systems

## Content
This directory groups tests by purpose:
- `unit/` for fast, deterministic tests that mirror the source structure.
- `integration/` for tests involving I/O, services, or multiple components.
- `manual/` for manually executed scripts that should not run in CI.
